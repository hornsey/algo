# ARTS
极客时间《左耳听风》ARTS活动打卡记录



**ARTS是什么？**

- **Algorithm**：每周至少做一个leetcode的算法题；
- **Review**：阅读并点评至少一篇英文技术文章；
- **Tip/Techni**：学习至少一个技术技巧；
- **Share**：分享一篇有观点和思考的技术文章。
